## Pl - GwentZSL

Projekt zaliczeniowy z przedmiotu technicznego w technikum. Jest to wariant gry karcienej GWENT, w którym występują postacie z mojego technikum. Projekt zawiera serwer napisany w Node.js i frontent w native JavaScript.

## Technologie

Do projektu wykorzystano język JavaSCript, server Node.js, bazę MongoDB oraz biblioteke do grafiki 3D Three.js.

## Status

:white_check_mark: Zakończony

Projekt został zakończony w czerwcu 2017 roku.

## EN - GwentZSL

A final project for a technical course at my technical school. This is a variant of the card game GWENT featuring characters from my technical school. The project includes a server written in Node.js and a frontend in native JavaScript.

## Technologies

The project uses JavaScript, a Node.js server, a MongoDB database, and the Three.js 3D graphics library.

## Status

:white_check_mark: Completed

The project was completed in June 2017.
